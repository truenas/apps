# UmlautAdaptarrEX

[UmlautAdaptarrEX](https://github.com/xpsony/UmlautAdaptarrEX) UmlautAdaptarrEX presents itself to the \*arrs as an indexer, sits between the \*arrs and the real indexers, and corrects both searches and results so that releases with umlauts or German titles are reliably found, downloaded and imported.
