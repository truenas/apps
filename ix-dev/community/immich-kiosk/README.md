# Immich Kiosk

Highly configurable slideshows for displaying [Immich](https://github.com/immich-app/immich) assets on browsers and devices.

## Configuration

Requires a running Immich instance. You will need:
- **Immich URL**: The URL of your Immich instance (e.g., `http://192.168.1.100:2283`)
- **Immich API Key**: Generate one in Immich under Account Settings > API Keys

Additional configuration can be done via environment variables with the `KIOSK_` prefix.
See [Immich Kiosk documentation](https://docs.immichkiosk.app) for all available options.
