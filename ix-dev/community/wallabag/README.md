# wallabag

wallabag is a read-it-later application: it saves a web page by keeping content only. Elements like navigation or ads are deleted.
It is a self-hostable alternative to services like Pocket.

This TrueNAS application integrates wallabag utilizing its built-in SQLite database.

## Notes

It is highly recommended to set the **Domain Name** (e.g. `https://wallabag.example.com`) during configuration to ensure internal links are generated correctly by the application.

For more information, see the [official wallabag documentation](https://doc.wallabag.org/).
