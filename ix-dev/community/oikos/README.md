# Oikos

[Oikos](https://github.com/ulsklyc/oikos) is a self-hosted, privacy-focused family planner. It bundles a shared calendar, tasks, shopping lists, meal planning, notes, contacts and budgeting into a single fast PWA — no cloud, no tracking.

On first launch, open the WebUI and create the initial admin account through the setup screen; no shell access is required.

Store the **Database Encryption Key** permanently if you enable encryption — it cannot be recovered, and losing it means losing access to your data.
