# Split-Pro

Split-Pro is an open source alternative to Splitwise for sharing expenses with friends and groups.

## Features

- Add expenses with individuals or groups
- Overall balances across groups
- Multiple currency support
- Upload expense bills
- PWA support
- Split expenses unequally (share, percentage, exact amounts, adjustments)
- Push notifications
- Download your data
- Import from Splitwise
- Simplify group debts
- Community translations
- Currency conversion
- Recurrent transactions (Beta)
- Bank account transaction integration (Beta)

## Authentication

**IMPORTANT:** You must configure at least ONE authentication method:
- Email/SMTP (recommended for simple setup)
- Google OAuth
- Authentik OAuth
- Keycloak OAuth
- Generic OIDC provider
