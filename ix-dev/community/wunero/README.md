# Wunero

[Wunero](https://github.com/XmlmXmlmX/wunero) is a shared wishlist application for collaborative gift planning.

Create wishlists, add items with automatic metadata extraction from URLs, and share them with friends and family.

## Features

- Create and manage multiple wishlists
- Automatic product info extraction from URLs
- Share wishlists with friends and family
- Mark items as purchased
- Email notifications via Resend (optional)
- Private and public wishlist options
