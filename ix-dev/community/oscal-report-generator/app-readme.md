# OSCAL Report Generator

Generate compliance documentation from OSCAL catalogs with AI-powered assistance.

## What is OSCAL Report Generator?

OSCAL Report Generator is a powerful tool designed to simplify the creation of compliance documentation. It helps organizations generate:

- **Statement of Applicability (SOA)**: Document which security controls apply to your system
- **System Security Plans (SSP)**: Comprehensive security documentation
- **Cloud Control Matrix (CCM)**: Cloud-specific security controls mapping

## Key Features

### AI-Powered Control Suggestions
Leverage artificial intelligence to automatically suggest relevant security controls based on your system description and requirements.

### Multiple Framework Support
- NIST 800-53 (Rev 4 and Rev 5)
- ISM (Australian Information Security Manual)
- IM8 (Hong Kong Government)
- And more frameworks

### Flexible Export Options
Export your compliance documentation in multiple formats:
- **JSON**: Machine-readable format
- **Excel**: Spreadsheet format for easy editing
- **PDF**: Professional documents for stakeholders
- **CCM**: Cloud Control Matrix format

### Modern Web Interface
User-friendly web application for managing your compliance documentation with an intuitive interface.

### RESTful API
Full API access for integration with other tools and automation workflows.

## Who Should Use This?

- **Compliance Officers**: Streamline compliance documentation
- **Security Teams**: Manage security controls across frameworks
- **IT Auditors**: Generate audit-ready documentation
- **Cloud Architects**: Map cloud controls to compliance requirements

## Getting Started

1. **Configure Storage**: Set up persistent storage for your configuration
2. **Access Web UI**: Click the Web Portal button to open the interface
3. **Select Framework**: Choose your compliance framework (NIST, ISM, etc.)
4. **Generate Reports**: Create SOA, SSP, or CCM documentation
5. **Export**: Download in your preferred format
