# OSCAL Report Generator

[OSCAL Report Generator](https://github.com/keekar2022/OSCAL-Reports) is a comprehensive compliance documentation tool for generating Statement of Applicability (SOA), System Security Plans (SSP), and Cloud Control Matrix (CCM) documents from OSCAL catalogs.

## Features

- **AI-Powered Control Suggestions**: Leverage AI to suggest relevant security controls
- **Multiple Framework Support**: NIST 800-53, ISM, IM8, and more
- **Multiple Export Formats**: JSON, Excel, PDF, and CCM formats
- **Modern Web Interface**: User-friendly interface for managing compliance documentation
- **RESTful API**: Programmatic access to all features

## Default Configuration

- **Default Port**: 30200
- **Container Port**: 3020
- **Health Check**: HTTP check on `/health` endpoint

## Getting Started

1. Install the app from the TrueNAS catalog
2. Configure storage path for persistent data
3. Access the web interface via the Web Portal button
4. Start creating your compliance documentation
