# Seafile

[Seafile](https://www.seafile.com/) - An open source cloud storage system with privacy preservation and teamwork features.

Seafile is a free, open-source, self-hosted cloud storage system with privacy preservation and teamwork features.
Collect, manage and collaborate around all your documents, photos, and videos easily.
Seafile focuses on reliability, performance and productivity.

## Features

- File syncing and sharing
- Team collaboration
- Privacy preservation
- Easy deployment
- Full-text search
- S3 compatible API
- And many more...

For more information, visit the [official Seafile documentation](https://manual.seafile.com/).
