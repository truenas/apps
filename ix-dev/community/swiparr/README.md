# Swiparr

Swiparr is a media discovery tool with a Tinder-like swipe interface. Browse movies and TV shows from Jellyfin, Emby, Plex, or TMDB — by yourself or together with friends.

Create sessions, swipe on what to watch, and let the matching algorithm find content everyone actually wants to see.

## Features

- Intuitive swipe interface for browsing media
- Multi-provider support: Jellyfin, Emby, Plex, or standalone TMDB
- Collaborative sessions with smart matching
- Guest lending for movie nights with friends
- Session controls (swipe limits, match strategy)
- Watchlist/favorites sync back to your media server
- Mobile-first PWA ready
